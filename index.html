Clase 1

comandos git

git init : Iniciar proyecto

git config --global user.name "Cristhian Zambrano" : configurar nombre de usuario

git config --global user.email johndoe@example.com : configurar correo de usuario

git add . : Agregar al stage area los cambios

git commit -m "comentario" : Crear un commit con el comentario respectivo

git status : Ver archivos que tienen cambios por guardar

git restore nombreArchivo : restaurar archivo a como estaba antes de enviarlos al stage area

git log : verificar los commits que tenenmos hasta el momento
